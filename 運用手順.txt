【新規ライブラリインストール時】
１，req.txtを出力
$ pip list
$ pip freeze > req.txt
２，リモートリポジトリにプッシュ

【別PCで同期時】
１，リモートリポジトリからプル
２，req.txtからpipでパッケージを入れる
$ pip install -r req.txt

【新規PCにクローン時】
１，venvで仮想環境作成
$ python -m venv PyEnv312
２，【同期時】の手順を実施
